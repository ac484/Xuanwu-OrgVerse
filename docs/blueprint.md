# **App Name**: OrgVerse

## Core Features:

- Secure Authentication: Authenticate users via Single Sign-On (SSO) with demo credentials (username: demo / password: 12345) for initial access.
- Global Context Switcher: A dropdown in the app's header lets users switch between organizations.
- Organization Creation: Enable users to create new organizations and define the permission structure.
- Permission Management: Implement a four-tiered permission system (Owner, Admin, Member, Guest) to manage access and control within each organization.
- Cross-Organization Collaboration Tool: Allow users to participate in projects across different organizations with a single identity, displaying clear visual labeling of external organizations.
- Container Creation: Enable top-level entities (users and organizations) to create logical containers for data and resources.
- Context-Aware UI Adaptation Tool: Generative AI model to detect organization context, then choose background colors that are consistent for each.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to evoke trust, authority, and openness.
- Background color: Light gray (#E0E0E0) to provide a clean and neutral backdrop.
- Accent color: Coral (#FF807A) for interactive elements, highlighting action items with a vibrant contrast.
- Body and headline font: 'Inter' for a clean, modern, and objective look.
- Use minimalist, professional icons representing various organizational functions and permissions.
- Employ a modular and stacking layout for logical containers, emphasizing clarity and ease of navigation.
- Implement smooth, subtle transitions between organizational contexts to ensure a seamless user experience.
